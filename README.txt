Конвертер для преобразования файлов типа '.csv' в файл типа '.json' или '.yaml'
В папке 'sourse' находится исходный код программы
Папка 'venv' хранит в себе необходимые библиотеки для корректной работы конвертора

Запуск конвертера.
- Открыть папку 'Конвертер'
- Переместить в папку файл с расширением '.csv'
- Запустить файл 'converter.exe'
- Ввести в поля требуемые значения
- В этой же папке создастся файл нужного вам типа, с полностью конвертируемой под стилистику данного типа информацией
